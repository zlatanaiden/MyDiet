# MyDiet 前端完整实施方案

## 技术栈

- **框架**: Vite + React 19 + TypeScript
- **样式**: Tailwind CSS v4（玻璃拟态主题）
- **路由**: React Router v7
- **图表**: Recharts（体重/体脂折线图）
- **动画**: Framer Motion（页面切换、礼花、火焰特效）
- **图标**: Lucide React（与设计稿一致）
- **特效**: canvas-confetti（打卡礼花）
- **字体**: Outfit（Google Fonts）

## 各模块实现要点

### 1. Homepage (`/`)

- **日历条**: 横向滚动的7天日期栏，当前日期品牌绿高亮，点击切换日期平滑过渡数据
- **环形进度**: SVG 环形图显示总热量 (1847/2200 kcal)，三条线性进度条 (蛋白质/碳水/脂肪)
- **三餐卡片**: 早/午/晚餐卡片，右侧勾选框，点击触发 `canvas-confetti` 礼花动画 + 进度条增长动画
- **打卡徽章**: 火焰图标 + "42 Days Streak"，点击切换实心/空心 + Framer Motion 火焰缩放特效 + 天数 +1
- **浮动工具条**: 右侧半透明垂直工具条（拍照识物、搜索）

### 2. Plan - 问卷 (`/plan`, 2-1)

- **表单卡片**: 玻璃卡片内含 Radio (目标)、Input (年龄/身高/体重/目标体重)、Select (性别/运动量)、Tags (忌口/过敏)
- **条件显示**: 选择"减肥/增肥"时动态展示目标体重输入
- **提交后**: 存入 Context + localStorage，路由跳转到 Dashboard

### 3. Plan - 仪表盘 (`/plan`, 2-2)

- **左侧边栏**: 用户状态卡 (身高/体重/BMI) + Current Goal + Restrictions，均可 Update
- **右侧主区**: 目标切换 Tab (Fat Loss/Gain Weight/Maintain) + 七天卡片流
- **每日卡片**: 早/午/晚餐缩略图 + 卡路里，刷新按钮随机换菜
- **点击日期**: 路由跳转 `/plan/day/monday`
- **动态刷新**: 修改目标/过敏源后，食谱区 Framer Motion fade+slide 刷新动画

### 4. Plan - 日详情 (`/plan/day/:day`, 2-3)

- **三大餐卡**: 大图 + 营养信息 (calories/protein/carbs/fats)
- **备选方案**: 每餐下方可展开 Alternatives 列表，Select 按钮替换当前菜品
- **Swap 按钮**: 随机替换当前菜品（动画过渡）

### 5. Identifier (`/identifier`, 3-1 + 3-2)

- **上传态 (3-1)**: 虚线框 Drop Zone，支持拖拽 + 点击上传，`onDragOver/onDrop` 事件
- **结果态 (3-2)**: 分屏布局，左侧预览图，右侧 AI 识别结果列表（食物名/重量/卡路里），可编辑修正
- **确认录入**: "Confirm & Log Today" 按钮，数据同步到 Homepage 的进度条（通过 Context）

### 6. Community (`/community`, 4-1 + 4-2 + 4-3)

- **标签切换**: Recommended (默认) / Trending，切换时瀑布流动画过渡
- **Recommended 视图 (4-1)**: 3列 CSS columns 瀑布流 + 右侧 Tags/Contributors 边栏
- **Trending 视图 (4-3)**: 4列无边栏瀑布流，顶部 "Last 24 Hours" 筛选，火焰热度标签
- **帖子卡片**: 图片 + 标题 + 用户 + 点赞，双击触发红心动画 (Framer Motion scale + opacity)
- **模态框 (4-2)**: 点击卡片弹出全屏模态，背景半透明模糊，左侧大图 + 右侧详情/评论，底部输入框
- **评论锚点**: 点击评论图标 -> `scrollIntoView` + `focus`
- **FAB**: 右下角 Refresh + Share Post 按钮

### 7. Profile (`/profile`)

- **Bio 区**: 头像 + 昵称 + UID
- **指标看板**: 4张数字卡片 (当前体重/目标体重/剩余天数/BMI)，BMI 带颜色标签
- **折线图**: Recharts `LineChart` 展示 30 天体重趋势，hover 显示 tooltip
- **偏好设置**: 过敏源标签 + 活动强度
- **社交摘要**: 帖子数/获赞/收藏
- **交互**: 内联编辑体重 (点击 pencil icon 弹出 input)、隐私 Toggle、单位切换 (kg/lb)

## 全局状态 (AppContext)

```typescript
interface AppState {
  user: UserProfile          // 个人信息
  nutrition: DailyNutrition  // 每日营养数据
  streak: number             // 打卡天数
  meals: WeeklyMeals         // 一周食谱
  planCompleted: boolean     // 是否已填写问卷
  unit: 'metric' | 'imperial' // 单位
}
```

所有状态持久化到 `localStorage`，刷新页面不丢失。
